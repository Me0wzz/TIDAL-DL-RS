# TIDAL-DL-RS [WIP]

---

Just working TIDAL Downloader written in Rust 

-----
## Features
- [x] Concurrent Download
- [x] Download Music (For now, FLAC only)
- [x] Support ID3 Tags
- [x] Support Track, Album, Playlist URL
